# Beginner Bioinformatics Using Public DNA Sequences

## Project Overview
This project is a beginner-friendly bioinformatics that analyzes DNA sequences from public or sample FASTA files.

## Features
- Reads FASTA DNA sequence files
- Calculates sequence length
- Calculates GC content
- Counts A, T, G, and C nucleotides
- Generates the reverse complement sequence

## Data Privacy Note
This project uses only public or sample DNA sequences. No employer, client, patient, proprietary, or confidential laboratory data were used.

## Why This Project Matters
DNA sequence analysis is a core skill in bioinformatics. This project helped me practice Python programming while connecting my microbiology laboratory background with computational biology.
